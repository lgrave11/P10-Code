Alliaceae J.G. Agardh (1858)
Batsch ex Borkh. (1797)[notes 1]

Allioideae is a subfamily of monocot flowering plants in the family Amaryllidaceae, order Asparagales. It was formerly treated as a separate family, Alliaceae.[6] The subfamily name is derived from the generic name of the type genus, Allium. It is composed of approximately eighteen genera.
